
# Exponentiation Operator (ES2016)

The Exponentiation Operator (**) is an operator for exponentiation

```javascript
{

    // ES5
    Math.pow(x, y);
    
    // ES6
    x ** y;
    
    // Example
    const twoTimesTwo = 2 ** 2; // 4
    const fourTimesFourTimesFour = 4 ** 3; // 64
}
```
